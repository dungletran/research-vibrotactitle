####################################################################################
#####			Guide for Arm Mapping Project			                
####################################################################################

1. Install:
 - Mingw mingw-get-inst-20111118
 - Qt SDK Qt_SDK_Win_offline_v1_1_4_en
 - qt-win-opensource-4.7.4-mingw
 <http://qt.nokia.com/downloads>

2. Build QExtSerialPort library <http://en.sourceforge.jp/projects/sfnet_qextserialport/downloads/qextserialport%20alpha/1.2win-alpha/qextserialport-1.2win-alpha.zip/>
- copy 2 files: "qextserialportd.dll", "libqextserialportd.a" in folder "\QExtSerialPort library\build" into "C:\Windows\System32"

3. Run: access file "QExtSerialPort library\Projects\MappingAnalysis\MappingAnalysis.pro"

Note:
1.pic file <500kb
2.debug by mingw complier
